## üìö Command Templates

### Template 1: Basic Analysis
```
Show me [metric] by [dimension]
Example: Show me revenue by product category
```

### Template 2: Time Series
```
Analyze [metric] trends over [time period]
Example: Analyze sales trends over the last 12 months
```

### Template 3: Comparison
```
Compare [metric] between [group A] and [group B]
Example: Compare conversion rates between mobile and desktop users
```

### Template 4: Top/Bottom
```
Show me top/bottom [N] [items] by [metric]
Example: Show me top 20 customers by lifetime value
```

### Template 5: Conditional
```
For [condition], show me [analysis]
Example: For orders above $500, show me average fulfillment time
```
